I am a systems programmer passionate about high-performance infrastructure, cloud-native security, and operating system architecture. My primary focus is on eliminating network bottlenecks by pushing logic out of user-space and directly into the Linux kernel.

- 🛡️ **Creator & Lead Maintainer of [@atrvasa](https://github.com/atrvasa)**
  I am currently architecting and building ATRVASA, a zero-overhead network observability and Zero-Trust firewall platform. It leverages the raw power of **eBPF (XDP/TC)** for kernel-level packet processing and **Rust (Aya framework)** for a memory-safe, ultra-fast control plane.

- ✍️ **Technical Writer & Creator**
  I believe in building in public. I regularly publish structured deep dives into system programming, Rust lifetimes, and Linux internals on my technical blog. I also run a dev-vlog documenting my day-to-day challenges in kernel development.

### ⚡ Quick Facts
- **Philosophy:** *"Bring logic to the data, not data to the logic."*
- **Why Rust?** Because I want fearless concurrency and memory safety without the cost of a garbage collector.
- **Why eBPF?** Because user-space context switches and `sk_buff` allocations are too expensive for modern 40G+ networks.
